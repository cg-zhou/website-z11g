## Features

- File sharing with expiry options
- Image processing tools
- Clipboard sharing
- Multi-language support
- Docker support

## Quick Start

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit http://localhost:3000 to see your application.
